# Student Grade Management System

## Project Description
This is a simple Java-based project designed to help teachers and students manage academic marks. It calculates the total percentage and automatically assigns a grade (A, B, C, etc.) based on the marks entered. It is a straightforward tool to automate result calculations instead of doing them manually.

## The Problem
Calculating grades for many students one by one is slow and prone to human error. It can be difficult to consistently track who passed or who performed exceptionally without a digital system.

## The Solution
This program takes the name and marks of a student as input. It then quickly calculates the average and determines the final grade. This makes the process faster, more accurate, and easy to read.

## Features
* **Input Marks**: Enter marks for any number of subjects.
* **Auto-Calculate**: Finds the average percentage automatically.
* **Grade Assignment**: Logic-based grading (e.g., A for 90+, B for 75+).
* **Final Report**: Displays a clean summary of the student's performance.

## Concepts Used
* **Classes & Objects**: To create a structured student profile.
* **Arrays**: To store and manage multiple subject marks.
* **Control Flow**: Using `if-else` logic to decide grades.
* **Scanner**: To handle user input from the console.
